Копируем файл проекта в удобное место
Заходим в папку проекта и копируем полный путь к содержимому папки
Вставляем этот путь в файлы startWork.bat и work-time.bat

В файл .env вставляем свою фамилию на латинице в поле LAST_NAME
При необходимости, там можно дописать выходные дни в поле NOT_WORKING_DAYS_OF_THIS_YEAR

Инициализируем проект (npm i) чтобы подкачать node_modules

Жмем WIN+R и вводим shell:startup
Жмем Enter
В эту папку перемещаем файл startWork.bat
При первом запуске ПК каждый день записывается время и далее отображается дата ухода

Консоль будет запускаться при вызове файла work-time.bat
При желании можно создать ярлык для его запуска и поместить его в панель задач (гугли сам)
